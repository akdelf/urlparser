## URLPARSER


Quickstart MVC router without regular. Routing: controller/action/params

Result array (
  'protocol',
  'domain',
  'method',
  'control',
  'action',
  'params'
)

* protocol (http, https)
* domain (default $_SERVER['QUERY_STRING'])
* method (GET, POST ...)
* permanent MVC rule: control/action/params
